@Rails Girls Guides

Install Rails

https://guides.railsgirls.com/install/linux
Поставились такие версии:
Should be SQLite 3.22.0 or higher: sqlite 3.45.1 2024-01-30 16:01:20 e876e51a0ed5c5b3126f52e532044363a014bc594cfefa87ffb5b82257ccalt1 (64-bit)
Should be Ruby 3.1.3 or higher: 3.1.3p185
Should be Rails 7.0 or higher: Rails 7.2.2.1


Создать директорию проекта:
rails new railsgirls
далее все команды вводить внутри неё
Start server for all:
rails server -b 0.0.0.0 -p 3000
Генерация всего необходимого для приложения:
rails generate scaffold idea name:string description:text picture:string
затем
rails db:migrate
и Ctr+C в терминале запущенного сервера, запустить снова
Если надо удалить сгенерённое:
rails destroy scaffold idea
