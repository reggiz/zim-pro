Git new

Created Четверг 27 февраля 2025


Создание и первая настройка.

git config --global user.name zumilko
git config --global user.email eprim@mail.ru
git init
это одноразовая команда, которую вы используете во время первоначальной
настройки нового репозитория, создаст новый подкаталог
.git в вашем текущем рабочем каталоге. 

Опционально:
git config --list
git config --global color.ui true
git config --global color.status auto
Если ругается, что уже тут был другой git:
git config --global --add safe.directory /run/media/data/Dropbox/zim/Notes/pro

Добавьте файлы в новый локальный репозиторий.
При этом они будут подготовлены для первой фиксации.
git add .
Чтобы отменить подготовку файла, используйте команду
git reset HEAD <file>
Из справки:

diff Show changes between commits, commit and working tree, etc
grep Print lines matching a pattern
log Show commit logs
show Show various types of objects
status Show the working tree status


Далее создание репо на github

Создаем удаленный репозиторий, указывая имя учетной записи.
curl -u 'zumilko' https://api.github.com/zumilko/repos -d'{"name":"zim-pro"}'
Вводим пароль от учетной записи:

Enter host password for user 'USER_NAME':
Репозиторий demo создан.
введите сюда описание изображения
Теперь выгружаем проект. 


git remote add origin https://github.com/zumilko/zim-pro.git
# Проверка (git config --list)
# Или измените текущий URL: 
git remote set-url origin https://github.com/zumilko/zim-pro.git

git push -u origin master

…or push an existing repository from the command line


git remote add origin git@github.com:zumilko/zim-pro.git
git branch -M main
git push -u origin main
